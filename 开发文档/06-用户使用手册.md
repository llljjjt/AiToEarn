## 快速开始

### 1. 安装客户端

#### Windows 用户
1. 双击 `AiToEarn-Setup-x.x.x.exe`
2. 按照安装向导完成安装
3. 安装完成后，桌面会出现 AiToEarn 图标

#### Mac 用户
1. 双击 `AiToEarn-x.x.x.dmg`
2. 将 AiToEarn 拖拽到 Applications 文件夹
3. 在启动台找到 AiToEarn 并打开

### 2. 首次登录

1. **启动应用**
   - 双击桌面图标启动 AiToEarn

2. **配置服务器地址**（仅首次需要）
   - 输入服务器地址：`http://your-server-ip:3000`
   - 或使用域名：`https://your-domain.com`
   - 点击"保存"

3. **登录账号**
   - 输入用户名和密码
   - 点击"登录"

4. **进入主界面**
   - 登录成功后，进入团队协作主界面

## 团队管理

### 查看团队信息

1. 点击左侧菜单 **"团队管理"**
2. 查看团队基本信息：
   - 团队名称
   - 团队描述
   - 创建时间
   - 成员数量

### 查看成员列表

在团队管理页面，可以看到所有成员信息：
- 成员姓名
- 邮箱
- 角色（Owner / Member）
- 加入时间

### 添加新成员（仅 Owner）

1. 点击 **"添加成员"** 按钮
2. 填写成员信息：
   - 用户 ID（从用户管理获取）
   - 用户名
   - 邮箱
   - 角色（选择 Member）
3. 点击 **"确定"**
4. 系统会发送飞书通知给新成员

### 编辑成员信息（仅 Owner）

1. 在成员列表中，点击 **"编辑"** 按钮
2. 修改成员信息（用户名、邮箱、角色）
3. 点击 **"确定"** 保存

### 移除成员（仅 Owner）

1. 在成员列表中，点击 **"删除"** 按钮
2. 确认删除操作
3. 成员将被移出团队

## 账号分组

### 为什么需要账号分组？

账号分组可以帮助你：
- **批量管理账号**：将同类账号归为一组
- **快速选择发布**：发布内容时一键选择整个分组
- **分工协作**：不同成员负责不同的账号组

### 查看分组列表

1. 点击左侧菜单 **"账号分组"**
2. 查看所有分组：
   - 分组名称
   - 描述
   - 账号数量
   - 创建时间

### 创建新分组

1. 点击 **"创建分组"** 按钮
2. 填写分组信息：
   - **分组名称**：例如 "抖音账号组"
   - **描述**：例如 "用于发布抖音内容的账号"
3. 点击 **"确定"**

### 添加账号到分组

1. 在分组列表中，点击 **"查看账号"** 按钮
2. 在右侧抽屉中，点击 **"添加账号"**
3. 从下拉框中选择要添加的账号（可多选）
4. 点击 **"确定"**

### 从分组移除账号

1. 在账号列表中，点击账号旁的 **"移除"** 按钮
2. 确认移除操作

### 编辑分组信息

1. 在分组列表中，点击 **"编辑"** 按钮
2. 修改分组名称或描述
3. 点击 **"确定"** 保存

### 删除分组

1. 在分组列表中，点击 **"删除"** 按钮
2. 确认删除操作
3. 注意：删除分组不会删除账号，只是解除分组关系

### 推荐的分组方式

**按平台分组**：
- 抖音账号组
- 小红书账号组
- 快手账号组
- B站账号组
- 公众号账号组

**按内容类型分组**：
- 产品宣传账号组
- 教程内容账号组
- 娱乐内容账号组

**按负责人分组**：
- 张三负责账号组
- 李四负责账号组

## 任务管理

### 查看任务列表

1. 点击左侧菜单 **"任务管理"**
2. 查看任务统计：
   - 总任务数
   - 进行中
   - 已完成
   - 待处理
3. 查看任务列表：
   - 任务标题
   - 负责人
   - 状态（待处理 / 进行中 / 已完成）
   - 优先级（低 / 中 / 高 / 紧急）
   - 截止时间

### 创建新任务

1. 点击 **"创建任务"** 按钮
2. 填写任务信息：
   - **标题**：例如 "发布产品宣传视频"
   - **描述**：详细说明任务内容
   - **负责人**：选择团队成员
   - **优先级**：选择 低/中/高/紧急
   - **截止时间**：选择日期
3. 点击 **"确定"**
4. 系统会发送飞书通知给负责人

### 筛选任务

使用顶部标签页筛选任务：
- **全部**：显示所有任务
- **待处理**：未开始的任务
- **进行中**：正在执行的任务
- **已完成**：已完成的任务

### 编辑任务

1. 在任务列表中，点击 **"编辑"** 按钮
2. 修改任务信息
3. 点击 **"确定"** 保存

### 分配任务

1. 在任务列表中，点击 **"分配"** 按钮
2. 选择新的负责人
3. 点击 **"确定"**
4. 系统会发送飞书通知给新负责人

### 完成任务

1. 在任务列表中，点击 **"完成"** 按钮
2. 任务状态变为"已完成"
3. 系统会发送飞书通知给团队

### 删除任务

1. 在任务列表中，点击 **"删除"** 按钮
2. 确认删除操作

### 任务协作流程示例

**场景：发布产品宣传视频**

1. **团队 Owner 创建任务**
   - 标题："发布产品宣传视频"
   - 描述："制作并发布新产品的宣传视频，需要发布到抖音、小红书、快手"
   - 负责人：张三
   - 优先级：高
   - 截止时间：本周五

2. **张三收到飞书通知**
   - "你有新任务：发布产品宣传视频"

3. **张三开始执行**
   - 制作视频内容
   - 选择"抖音账号组"（3个账号）
   - 一键发布到 3 个抖音账号

4. **张三完成任务**
   - 点击"完成"按钮
   - 系统发送飞书通知："张三完成了任务：发布产品宣传视频"

5. **团队查看数据**
   - 在数据看板查看发布效果

## 内容发布

### 发布图文内容

1. 点击左侧菜单 **"内容管理"**
2. 点击 **"创建内容"** 按钮
3. 选择 **"图文"** 类型
4. 填写内容信息：
   - **标题**：内容标题
   - **正文**：内容正文（支持富文本）
   - **封面图**：上传封面图片
   - **标签**：添加标签（用逗号分隔）
5. 选择发布平台：
   - ☑ 小红书
   - ☑ 公众号
6. 选择账号：
   - 方式 1：选择账号分组（推荐）
   - 方式 2：手动选择单个账号
7. 点击 **"发布"** 按钮
8. 等待发布完成（可在发布记录查看进度）

### 发布视频内容

1. 点击左侧菜单 **"内容管理"**
2. 点击 **"创建内容"** 按钮
3. 选择 **"视频"** 类型
4. 填写内容信息：
   - **标题**：视频标题
   - **描述**：视频描述
   - **视频文件**：上传视频（支持 MP4, MOV, AVI）
   - **封面图**：上传封面图片
   - **标签**：添加标签
5. 选择发布平台：
   - ☑ 抖音
   - ☑ 快手
   - ☑ B站
6. 选择账号分组：
   - 例如："抖音账号组"（3个账号）
7. 点击 **"发布"** 按钮
8. 系统会自动：
   - 上传视频到各平台
   - 发布到选中的所有账号
   - 发送飞书通知

### 查看发布记录

1. 点击左侧菜单 **"发布记录"**
2. 查看所有发布记录：
   - 内容标题
   - 发布平台
   - 发布账号
   - 发布状态（成功 / 失败）
   - 发布时间
3. 点击记录可查看详情：
   - 浏览量
   - 点赞数
   - 评论数
   - 分享数

### 批量发布技巧

**使用账号分组批量发布**：
1. 提前创建好账号分组（例如："抖音账号组"包含 5 个账号）
2. 发布内容时，选择"抖音账号组"
3. 一键发布到 5 个账号，节省时间

**跨平台发布**：
1. 同一内容可以同时发布到多个平台
2. 例如：同时勾选"抖音"、"快手"、"B站"
3. 系统会自动适配各平台的格式要求

## 数据看板

### 查看数据概览

1. 点击左侧菜单 **"数据看板"**
2. 查看核心指标：
   - **总发布量**：团队累计发布的内容数量
   - **总浏览量**：所有内容的累计浏览量
   - **总互动数**：点赞 + 评论 + 分享的总数
   - **平均互动率**：互动数 / 浏览量

### 切换时间范围

使用顶部单选按钮切换：
- **日报**：查看今日数据
- **周报**：查看本周数据
- **月报**：查看本月数据

### 查看平台数据对比

在"平台数据对比"表格中，可以看到：
- 各平台的发布量
- 各平台的浏览量
- 各平台的点赞数
- 各平台的评论数
- 各平台的分享数

**分析技巧**：
- 找出表现最好的平台，加大投入
- 找出表现较差的平台，分析原因

### 查看趋势图表

（预留位置，后续集成 ECharts）

可以看到：
- 浏览量趋势（折线图）
- 互动数趋势（折线图）
- 发布量趋势（柱状图）

### 刷新数据

点击右上角 **"刷新"** 按钮，获取最新数据。

### 数据分析示例

**场景：分析本周内容表现**

1. **切换到周报**
   - 点击"周报"标签

2. **查看核心指标**
   - 本周发布量：20 条
   - 本周浏览量：50,000
   - 本周互动数：2,500
   - 平均互动率：5%

3. **查看平台对比**
   - 抖音：发布 10 条，浏览 30,000，互动 1,800（互动率 6%）
   - 小红书：发布 5 条，浏览 15,000，互动 600（互动率 4%）
   - 快手：发布 5 条，浏览 5,000，互动 100（互动率 2%）

4. **得出结论**
   - 抖音表现最好，应加大投入
   - 快手表现较差，需要优化内容策略

5. **采取行动**
   - 下周增加抖音发布量到 15 条
   - 分析快手失败原因，调整内容方向

## 常见问题

### 1. 无法连接服务器

**问题**：客户端显示"网络错误"或"无法连接服务器"

**解决方法**：
1. 检查服务器地址是否正确
2. 检查网络连接是否正常
3. 联系管理员确认服务器是否启动
4. 尝试重启客户端

### 2. 登录失败

**问题**：输入用户名密码后，提示"登录失败"

**解决方法**：
1. 检查用户名和密码是否正确
2. 确认账号是否已被添加到团队
3. 联系团队 Owner 确认账号状态
4. 尝试重置密码

### 3. 发布内容失败

**问题**：点击"发布"后，显示"发布失败"

**可能原因**：
- 平台账号未授权或授权过期
- 视频格式不支持
- 视频大小超过限制
- 网络连接中断

**解决方法**：
1. 检查账号授权状态，重新授权
2. 确认视频格式（支持 MP4, MOV, AVI）
3. 确认视频大小（< 500MB）
4. 检查网络连接
5. 查看发布记录中的错误详情

### 4. 看不到数据

**问题**：数据看板显示"暂无数据"

**可能原因**：
- 还没有发布过内容
- 数据采集任务未运行
- 平台 API 限流

**解决方法**：
1. 确认已发布内容
2. 等待数据采集（每小时自动采集一次）
3. 点击"刷新"按钮手动刷新
4. 联系管理员检查数据采集任务

### 5. 飞书通知收不到

**问题**：任务分配或完成后，没有收到飞书通知

**可能原因**：
- 飞书机器人未配置
- Webhook URL 错误
- 飞书机器人被禁用

**解决方法**：
1. 联系管理员检查飞书机器人配置
2. 确认飞书群中有机器人
3. 测试发送通知

### 6. 账号授权失效

**问题**：发布时提示"账号授权已过期"

**解决方法**：
1. 点击左侧菜单 **"账号管理"**
2. 找到失效的账号
3. 点击 **"重新授权"** 按钮
4. 按照提示完成授权流程

### 7. 视频上传慢

**问题**：上传视频时进度条很慢

**解决方法**：
1. 检查网络速度（建议 > 10Mbps）
2. 压缩视频大小（推荐 < 100MB）
3. 避免高峰时段上传
4. 使用有线网络代替 WiFi

### 8. 客户端卡顿

**问题**：客户端使用时卡顿或响应慢

**解决方法**：
1. 关闭其他占用内存的程序
2. 重启客户端
3. 清理客户端缓存
4. 升级电脑内存（推荐 > 8GB）

## 最佳实践

### 1. 团队协作

- **明确分工**：每个成员负责特定的账号分组
- **及时沟通**：使用飞书群及时沟通任务进度
- **定期复盘**：每周查看数据看板，总结经验

### 2. 内容发布

- **提前规划**：每周一规划本周发布内容
- **批量操作**：使用账号分组批量发布，提高效率
- **错峰发布**：避开平台流量高峰，提高曝光率

### 3. 数据分析

- **每日查看**：每天查看日报，了解最新数据
- **每周总结**：每周五查看周报，总结本周表现
- **每月复盘**：每月初查看月报，制定下月计划

### 4. 账号管理

- **定期检查**：每周检查账号授权状态
- **及时续期**：授权快过期时及时重新授权
- **分组管理**：合理创建账号分组，方便批量操作
